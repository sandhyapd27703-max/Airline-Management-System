# Airline Management System

A Java-based **Airline Management System** built using **Swing** for the GUI and **MySQL** for database management.  
This project allows users to manage flights, passengers, bookings, and cancellations easily.

---

## Features
-  Login System** for admin access  
-  Add and Manage Passengers**  
-  Book Flights and Generate PNR**  
-  View Journey Details**  
-  Cancel Tickets**  
-  Generate Boarding Pass**

---

##  Technologies Used
- Java (Swing, AWT)**
- MySQL Database**
- JDBC (Java Database Connectivity)**
- NetBeans IDE**
